# 音乐小程序

> A Mpvue project

## 快速开始

``` bash
# 安装依赖
npm install

# 启动服务
npm run dev

# 打包项目
npm run build
```

## 技术栈

- Mpvue
- Vue
- Node.js
- puppeteer
- Sass
